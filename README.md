# MBR-Overwrite

A basic console application to overwrite MBR records, thus rendering the client device unbootable.

## Requirements

Requires admin privileges to run.

## Works on
Windows 11 ❓
Windows 10 ✔️
Windows 8.1 ✔️
Windows 7 and before ✔️
